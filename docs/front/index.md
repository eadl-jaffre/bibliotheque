# Frontend

## Stack technique

- Angular
- TypeScript
- Bootstrap

## Démarrage local

Depuis la racine du projet:

```bash
make run-front
```

Application locale:

- http://localhost:4200

## Tests frontend

### Unitaires (Mocha)

```bash
cd front
npm run test:mocha
```

### E2E (Mocha + API réelle)

```bash
cd front
npm run test:e2e
```
